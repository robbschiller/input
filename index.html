Input

Experimenting with form and function on the input element.

How many variations of the input can be formed with just one selector for each the label and input tags? This little project is an experiement on one of the webs most important points of interaction.

HTML

								<label>Input</label>
								<input type="text">

CSS

								.label {
									background: blue;
								}
